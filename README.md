# Smeriwether Everydollar

## How do I install these formulae?

`brew install smeriwether/everydollar/<formula>`

Or `brew tap smeriwether/everydollar` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "smeriwether/everydollar"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
